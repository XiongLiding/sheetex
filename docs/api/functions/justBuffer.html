    Function justBuffer

    • justBuffer(
          data: DataRow[] | JustData,
          colWidths?: number[],
      ): Promise<Uint8Array<ArrayBufferLike>>

      快速生成工作簿 Buffer

      根据用户提供的数据以及列宽配置，生成一个带有默认样式的 Excel 文件并得到相应的 Buffer

      Parameters

      • data: DataRow[] | JustData

        表格数据

      • colWidths: number[] = []

        列宽

      Returns Promise<Uint8Array<ArrayBufferLike>>

      • 生成的 Excel 文件的 Buffer

      Remarks

      默认样式是指：

      1. 标题加大加粗合并单元格居中显示
      2. 表头加粗居中显示
      3. 数据居中显示
      4. 所有内容加上黑色细线边框

      Example

      数据

      const data = [
        [1, '张三', 100],
        [2, '李四', 90],
        [3, '王五', 80],
      ];
      1. 仅数据
      const buffer = await justBuffer(data);
      1. 指定标题、表头并设置列宽
      const buffer = await justBuffer({
        caption: '成绩单',
        header: ['学号', '姓名', '成绩'],
        data,
      }, [20, 20, 20]);
